This folder contains 6 header files. Brief descriptions are below:

1) REF_LU_config.h: Defines constants, includes common headers, gives author info etc

2) Generate_Matrix.h: Generates dense matrices randomly

3) Output.h: Functions for printing output onto the screen, into files, and final memory management

4) REF_LU_Options.h: Defines the REF_LU_Options struct and contains functions relating to reading in info from command line

5) REF_LU.h: Defines the REF_mat struct and contains functions and routines related to REF LU factorization and Q_mat

6) QLU.h: Defines the QLU_mat struct and contains functions related to Crout and Doolittle factorization
